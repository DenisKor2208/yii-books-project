# Тестовый каталог книг на Yii1

Простой тестовый веб-каталог книг с возможностью подписки на новых книги авторов и отправкой SMS-уведомлений (эмулятор).  
Реализовано разграничение доступа: гость (только просмотр и подписка) и авторизованный пользователь (полный CRUD).

## Быстрый старт

### 1. Клонировать репозиторий

```bash
git clone https://github.com/DenisKor2208/yii-books-project.git
cd project
```

### 2. Настройка ENV

Скопируйте файл .env.example в .env и при необходимости отредактируйте параметры подключения к БД и API_KEY для отправки смс

```bash
cp .env.example .env
```

### 3. Запустить контейнеры

```bash
make up
```

### 4. Установить зависимости Composer

Запустите установку внутри контейнера php-cli:

```bash
make install
```

### 5. Применить миграции

```bash
make migrate
```

### 6. Настроить права на папки для загрузки

```bash
docker compose exec php-fpm chmod -R 777 /var/www/public/uploads/books
```

### 7. Проверить работу приложения

- Откройте в браузере: http://localhost:8080

### Доступ к дополнительным сервисам

- phpMyAdmin: http://localhost:8081
- Логин: root, пароль: secret (или тот, что указан в .env).

### Структура проекта и особенности

- Роли пользователей:
  - Гость (неавторизованный) – может просматривать книги и авторов, подписываться на новых книги автора (ввод номера телефона на странице автора).
  - Авторизованный пользователь – полный CRUD для книг и авторов.

- SMS-уведомления реализованы через эмулятор smspilot.ru. Реальной отправки нет, все попытки логируются в protected/runtime/application.log.

- Загрузка обложек – файлы сохраняются в public/uploads/books/.

- Миграции находятся в protected/migrations/
